# Introduction


When user clicks on the help icon ![image2013-2-8 10:19:28.png](/.attachments/29920096.png)


at the left of the menu bar, the screen's user guide opens. 
By providing the Phoenix URL and Wiki URL, Wiki Redirect will link up the related Wiki user guide to the respective Phoenix screen. Thus allowing user to directly view the screen's user guide when they click on the help icon.



### How to get there?




::: mermaid
graph LR
A("ADMINISTRATION")-->0("SYSTEM")
0-->1("ADD WIKI REDIRECT")

:::
**Screen Activity** 
Wiki Redirect enables user to perform the following activity:

- Create, view, update and delete Wiki Redirects

PreconditionThe user guide in Wiki and the Phoenix screen must exist so that administrators can grab their URL for the link up
