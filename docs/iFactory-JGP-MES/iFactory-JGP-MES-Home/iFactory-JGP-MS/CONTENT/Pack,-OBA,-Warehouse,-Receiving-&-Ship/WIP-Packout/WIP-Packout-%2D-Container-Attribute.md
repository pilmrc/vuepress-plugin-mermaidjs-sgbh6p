# Introduction

Container Attribute Assignment allows user to insert non-retained value or retained value for a container during WIP packout process. 
**Pre-condition** 
[Container attribute assignment](/iFactory-JGP-MES/iFactory-JGP-MES-Home/iFactory-JGP-MS/CONTENT/General-Production/Attribute-Assignment.md)
need to be configured before can insert container attribute at WIP packout workstation.**Behavior at Runtime** 
Container Attribute Assignment will appear as soon as the system identified the container type has Container Attribute Assignment configured. Once user press "Close" button / container auto close , then the container attribute values will be captured at Container view and container maintenance.

1.Configure container attribute assignment .
![image2019-2-4_12-40-5.png](/.attachments/42991807.png)


2.  Container Attribute Assignment is shown when the selected container type has attribute assignment configured.
![image2019-2-4_12-43-35.png](/.attachments/42991808.png)


3.  Container attribute assignment need to be completed before allowing user to scan a WIP for packing.
![image2019-2-4_12-48-39.png](/.attachments/42991809.png)


![image2019-2-4_12-49-28.png](/.attachments/42991810.png)


4.  Container Attribute values will be assigned and captured once the container is closed. Container Attribute value can view via Container View and Container Maintenance.
![image2019-2-4_12-51-15.png](/.attachments/42991811.png)


![image2019-2-4_12-55-5.png](/.attachments/42991812.png)



5. The Retained value will be remained if the container type is the same else it will be cleared off from the screen.
![image2019-2-4_12-56-4.png](/.attachments/42991813.png)



6. When user scans an existing container number or packed WIP that has container attribute, system retrieves and displays the container attribute(s). This includes retained and non-retained values.
![image2019-2-4_12-56-4.png](https://jblprd.visualstudio.com/3416e237-a20a-4514-985d-7993adabea7d/_apis/wit/attachments/48b2210f-104a-45b2-b998-fd8069d10dcb?fileName=image.png)
