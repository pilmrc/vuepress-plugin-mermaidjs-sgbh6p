# Introduction

In the IFactory API, a few functions are provided to retrieve WIP information commonly used as critical input to manufacturing processes. These functions can be called anytime after a successful login



### Retrieve WIP



The Retrieve WIP function returns all the most fundamental information about a WIP. This includes the sources of its vital data such as serial number, its source of composition such as the material and the BOM, its current status and etc. One of the most important info from this
